# 📊 BodiTech-Lecture: 주간 보고서 자동 취합 및 병합 프로그램

아웃룩(Outlook) 메일을 검색하여 각 부서에서 송부한 주간 회의자료(PPT)를 자동으로 다운로드하고, 미제출 부서를 확인한 뒤, 하나의 PPT 파일로 병합하는 파이썬 자동화 스크립트입니다.

## ✨ 주요 기능 (Features)
1. **특정 조건 파일 수집**: 오늘 날짜(예: `260521`)와 특정 키워드(`주간회의자료`, `회의자료`)가 포함된 PPT/PPTX 파일만 정확히 필터링하여 다운로드합니다.
2. **미제출 부서 트래킹**: 사전에 정의된 부서 및 담당자 명단을 바탕으로 메일을 보내지 않은 누락 부서를 확인하고 콘솔에 알림을 띄워줍니다.
3. **자동 PPT 병합**: 수집된 여러 개의 PPT 파일을 하나의 파일(`주간보고병합.pptx`)로 이어 붙여 자동으로 생성합니다.

---

## ⚠️ 사전 요구 사항 (Prerequisites)
이 스크립트는 윈도우(Windows) 환경의 COM 객체를 제어하므로 아래 조건이 충족되어야 합니다.
* **OS**: Windows 운영체제
* **Microsoft Outlook**: 데스크톱 앱이 설치되어 있고 계정이 로그인된 상태
* **Microsoft PowerPoint**: 데스크톱 앱 설치 필수
